HTTP-XML-API на NodeJS

Шаблон POST-запроса через встроенный fetch в NodeJS 18+. Замените учетные данные, отправителя, номер получателя и текст сообщения.

Все примеры

Язык скрипта

PHP Java Python NodeJS

NodeJS script

const apiUrl = 'https://smspro.nikita.kg/api/message';

const xml = `<?xml version="1.0" encoding="UTF-8"?>
<message>
    <login>USERNAME</login>
    <password>PASSWORD</password>
    <sender>NIKITA</sender>
    <text>Test message from HTTP-XML-API</text>
    <phones>
        <phone>996700000000</phone>
    </phones>
</message>`;

const response = await fetch(apiUrl, {
    method: 'POST',
    headers: {
        'Content-Type': 'application/xml; charset=UTF-8',
    },
    body: xml,
});

const body = await response.text();

console.log('HTTP status:', response.status);
console.log(body);

Важно: структура XML в этом шаблоне оставлена как стартовая заготовка. Для боевого подключения сверьте поля с актуальным описанием XML-API.
